## Quick Reference

Vector databases store high-dimensional vectors (embeddings) generated by ML models. They enable semantic search by finding vectors that are "closest" to a query vector in the vector space. Popular choices include Pinecone, Chroma, Qdrant, Milvus, and Weaviate.
