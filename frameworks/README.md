| Svelte | Compiler-based reactive framework (4/5) | [svelte/](svelte/) |
| Next.js | React meta-framework with SSR/SSG (14+) | [nextjs/](nextjs/) |
| Nuxt | Vue meta-framework (3) | [nuxt/](nuxt/) |

## Quick Reference

```
# Frontend
npm create vite@latest my-app -- --template react
npx create-next-app@latest my-app
npx nuxi@latest init my-app

# Backend
pip install fastapi uvicorn
composer create-project laravel/laravel my-app
cargo install cargo-generate && cargo generate --git actix/actix-extras
```
