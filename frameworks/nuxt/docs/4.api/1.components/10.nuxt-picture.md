# <NuxtPicture>

## Introduction

`` is a drop-in replacement for the native `<picture>` tag.

Usage of `` is almost identical to [``](/docs/3.x/api/components/nuxt-img) but it also allows serving modern formats like `webp` when possible.

Learn more about the [`<picture>` tag on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/picture).

## Setup

In order to use `` you should install and enable the Nuxt Image module:

```bash [Terminal]
npx nuxt module add image
```
