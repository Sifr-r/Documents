# Content loader returned an entry with an invalid id.

## Introduction

> **Example error message:**<br/>
The content loader for the collection **blog** returned an entry with an invalid `id`:<br/>
&#123;<br/>
  "id": 1,<br/>
  "title": "Hello, World!"<br/>
&#125;

## What went wrong?

A content loader returned an invalid `id`.
Make sure that the `id` of the entry is a string.
See the [Content collections documentation](/en/guides/content-collections/) for more information.
