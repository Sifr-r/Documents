# Fringe Cases

## Warnings & Notes

> [!NOTE]
> Nesting multiple `<h1>` elements in nested [sectioning elements](/en-US/docs/Web/HTML/Reference/Elements#content_sectioning) was allowed in older versions of the HTML standard. However, this was never considered a best practice and is now non-conforming. Read more in [There Is No Document Outline Algorithm](https://adrianroselli.com/2016/08/there-is-no-document-outline-algorithm.html).
