# Fringe Cases

## Warnings & Notes

> [!NOTE]
> `rotateZ(a)` is equivalent to
> `rotate(a)` or
> `rotate3d(0, 0, 1, a)`.

> [!NOTE]
> Unlike rotations in the 2D plane, the composition of 3D rotations is usually
> not commutative. In other words, the order in which the rotations are applied impacts the result.
