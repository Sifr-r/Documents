# Fringe Cases

## Warnings & Notes

> [!NOTE]
> The [CSS writing modes](/en-US/docs/Web/CSS/Guides/Writing_modes) module defines a `digits <integer>` value for the `text-combine-upright` property to display two to four consecutive {{Glossary("ASCII")}} digits (U+0030–U+0039) such that it takes up the space of a single character within the vertical line box, however, this is not supported in any browsers.
