# Fringe Cases

## Warnings & Notes

> [!NOTE]
> [WCAG 2.1 Success Criterion 1.3.5: Identify Input Purpose](https://www.w3.org/WAI/WCAG21/Understanding/identify-input-purpose.html) does not require that autocomplete or autofill features must work; it only requires that form fields that relate to specific personal user information are programmatically identified. This means that the criterion can still be met (by adding the relevant [`autocomplete`](/en-US/docs/Web/HTML/Reference/Attributes/autocomplete) attributes to individual form fields) even when autocompletion for the form itself has been turned off.
