# Fringe Cases

## Warnings & Notes

> [!NOTE]
> In [multi-column layout](/en-US/docs/Web/CSS/Guides/Multicol_layout), which is a fragmented context, the sizing returned by `ResizeObserverSize` will be the size of the first column.

> [!NOTE]
> For more explanation of writing modes and block and inline dimensions, read [Handling different text directions](/en-US/docs/Learn_web_development/Core/Styling_basics/Handling_different_text_directions).
