> [!NOTE]
> Before Firefox 118, the <kbd>⊞ Windows</kbd> key was handled
> as an "OS" key rather than the "Meta" key. `KeyboardEvent.metaKey` was
> `false` when the <kbd>⊞ Windows</kbd> key was pressed.
